# ATMO Engine

A lightweight C++ game engine with Luau scripting and ECS architecture.


# Requirements

## Windows
- C++23 compiler (MSVC)
- xmake

## MacOs
- C++23 compiler (clang++)
- xmake

## Linux
- c++23 compiler (g++)
- xmake

Install dependencies:
https://xmake.io/guide/quick-start.html


# Build
```
git clone git@github.com:ATMO-Engine/atmo.git
git submodule update --init --recursive
xmake
```
